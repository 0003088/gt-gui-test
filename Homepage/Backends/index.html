Backends

A very important part of libelektra are the kdb* functions. They are provided through backends. They take care that the key and keysets are read and written to the database.

There could be an implementation for every surporse. Berkleydb or fast in-ram hashes for embedded up to sql server requests for thousends of clients. Its up to you: The backends will immadiately work with elektrified software.

Backends

  • Filesys backend by Avi. It is a very secure backend with all features that your filesystem supports.
  • Berkley DB by Avi. It is a fast backend with small footprint.
  • Ini Db by Markus. It is a compromise between efficient storage, editable by hand and fast access.

All people working on the backends can be found on the Elektra mailing list.
