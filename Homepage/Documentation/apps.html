Tools and Software Patches

  • Regedit GUI to edit the key database, using the QT library, by Gregor Burger. There is a Berlios entry with Subversion repository, and RPM packages available.
  • Elektrified X.org [see README] to use a key database instead of xorg.conf file.
  • Elektrified /sbin/init and Fedora Core 2 RPMs [see README] for the original sysvinit package from Miquel van Smoorenburg.
  • Users and Groups Name System Module for GLibC, by Jens Andersen. This is a module that can be activated by the name system switch mechanism (/etc/nsswitch.conf) to let users and groups databases be Elektrified, instead of /etc/{passwd|shadow|groups}.
  • Daniel Lambea and Israel Bethencourt are writing patches for basic system tools, and will have soon a full Elektrified Linux distribution.
