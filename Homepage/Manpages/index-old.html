1.1. Online Manual Pages and API Documentation

Elektra man pages can be accessed online:

  1. kdb.1: The kdb command man page. Describes the usage of the command used to administer Elektra's key database. You don't need it if you are already reading this.
  2. elektra.7: Discussion about what is the Elektra Initiative etc. You don't need it if you are already reading this.
  3. elektra.5: Elektra internals.
  4. kdb.3: C API documentation for the KeyDB class. Browse the doxygen instead.
  5. key.3: C API documentation for the Key class. Browse the doxygen instead.
  6. keyset.3: C API documentation for the KeySet class. Browse the doxygen instead.

You can also browse or download the API documentation (also provided in the -devel RPM), by Doxygen. It is complete, with many code examples.
