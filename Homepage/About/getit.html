Elektra was formerly named Linux Registry, so many links will still take you to "Linux Registry" named pages. They'll change over time.

The Elektra Initiative software and info can be found at elektra.sf.net and (in all) the following places:

  • C source and RPMs from sourceforge.net
  • Subversion repository, thanks to Sean Russel
  • Wiki space, made available by Daniel Lambea
  • Freshmeat project page
  • Advogato project page
  • Orkut Community
  • Many Elektra developers use to get together in the #elektra channel of the irc.freenode.net IRC server
  • Subscribe to general discussions list or see the list archives

There is an OpenOffice.org presentation [HTML without animations] that anybody can deliver in community events, to help create awareness about the Elektra Initiative. There is also a german translation by Markus Raab.

To get the last snapshot from Subversion do the following: 

bash:~$ cd src
bash:~/src$ svn co http://germane-software.com/repositories/elektra/trunk elektra
bash:~/src$ cd elektra
bash:~/src/elektra$ # files are here

Elektra has a BSD license.
