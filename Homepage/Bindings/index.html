Language Bindings and Packages

  • Ruby bindings, by Sean Russel. If you are writing code in Ruby, and need an easy way to store/retrieve configuration/state data, this is for you.
  • Lua bindings, by Rob King. Check also its README file.
  • JRegistry, by Alexopoulos Charalampos, are Java bindings for the Elektra API, and apparently a GUI for Elektra administration.
  • Fedora Core RPM packages
  • Gentoo ebuild, by Martin Holzhauer
  • Arch Linux PKGBUILD, by Lukas Sabota, and also the Arch Linux binary packages.

All people working on that can be found on the Elektra mailling list.
